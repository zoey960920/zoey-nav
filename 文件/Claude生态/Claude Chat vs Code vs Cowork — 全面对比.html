Claude Chat

对话即答案 · 浏览器 / App

经典对话界面，问什么答什么。适合提问、文档分析、头脑风暴、写作辅助。你问一句，它答一句——主动权在你手上。

  • 百万 token 上下文窗口
  • 网页搜索 + 文档/图片分析
  • 6000+ MCP 应用集成

Claude Code

终端里的编程搭档 · CLI / IDE

在终端里运行，自动理解整个代码库。给它一个需求，它独立完成：读 issue → 写代码 → 跑测试 → 修 bug → 提 PR。

  • 全代码库智能搜索与理解
  • 多文件协同编辑
  • Git 全流程自动化
  • VS Code / JetBrains / Slack 集成
  • 本地运行，代码不离开你的机器
🤝

Claude Cowork

桌面上的AI同事 · 桌面应用

把 Code 的自主能力带给非技术人员。用自然语言描述任务，Claude 就能操作本地文件，自主完成复杂知识工作。

  • 本地文件读写与管理
  • 自动生成报告、演示文稿、备忘录
  • 周期性任务自动化
  • 手机提交任务，电脑执行
  • 关键决策前暂停确认

⌨️ 用 Code

⌨️ 用 Code

批量处理本地文件、提取信息并整理输出——Cowork 的优势场景。
工作流

一个任务，三种体验

假设你要"整理本月客户反馈并生成分析报告"：

💬 Chat 的体验

① 上传反馈数据

② Claude 给出分析

③ 追问："按产品线分类？"

④ 继续："做成表格"

⑤ 复制结果到文档

~10 轮对话，全程在场

⌨️ Code 的体验

① "读取 ./feedback 下所有 CSV，按产品线分类分析，生成 report.md"

② 自动读取、写脚本、生成报告

③ 自动 commit

1 条指令，全自动完成

🤝 Cowork 的体验

① "整理客户反馈文件夹，做分析报告"

② Cowork 开始工作

③ 中途问你："3 个重复文件，合并还是跳过？"

④ 交付格式化好的报告文件

喝杯咖啡，回来就好了
