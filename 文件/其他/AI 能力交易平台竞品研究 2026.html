01 · PLATFORM ANALYSIS
🌊 FlowGPT
flowgpt.app · 原 flowgpt.com（已弃用）
$14.99
Plus 月费
AI 角色 / Roleplay Bot 广场。从 Prompt 工具库转型为 AI 伴侣平台，1M+ Bots，目标用户从开发者变为普通消费者，核心价值主张是"和 AI 角色对话"。
产品结构
左侧栏：Explore / Leaderboard / Bounty / Wallet
Tab 分类：Job Hunting / Health / Academic / Business / Programming
三维筛选：Trending × All style × All gender
Banner 广告：推自有 App（Emochi / Branches）
商业模式
Plus $14.99/月：1500 Flux，解锁 5 Featured Models
Ultra $24.99/月：2500 Flux，6× AI Memory，+10% 充值奖励
Flux = 平台虚拟货币（按对话消耗）
Bounty 悬赏机制：用户给创作者出题
优势 / 短板
优势①：Mass content（1M+ Bots），长尾覆盖广
优势②：Leaderboard + Bounty 创造竞争激励
优势③：性别/风格筛选 → 精准匹配偏好
短板①：内容质量良莠不齐，缺乏审核
短板②：站点稳定性差，网络错误频发
短板③：Prompt 属性弱化，创作者变现不透明

+ Create 按钮引导用户成为创作者
无广告，无会员特权内容

优势③：Builder 工具降低创作门槛

🎨 Civitai
civitai.com（SFW）+ civitai.red（NSFW）· 2026.04 拆分
3色 Buzz
虚拟货币体系 2025.10 重构
全球最大 AI 图像模型社区。以 LoRA / Checkpoint / VAE 等模型为核心交易品，Buzz 虚拟货币驱动创作者经济。2025 年支付危机（信用卡全停）后转向加密货币，2026 年平台一分为二。
Buzz 经济体系
黄色 Buzz 加密货币购买，点对点，无订阅
蓝色 Buzz 免费获取：日签 / 社区互动 / 创作奖励
绿色 Buzz SFW 内容生成专用
Creator 补偿池：每月从平台营收中按比例拨款
Supporter 会员：~$5-10/月
内容 & 社区
模型类型：Checkpoint / LoRA / Hypernetwork / VAE / Workflow
每个模型附：样本图 + Prompt + 参数 + Seed
社区极强：评分、评论、"Run on Civitai"按钮
内容分级（SFW/NSFW）是核心争议来源
优势 / 危机
优势①：最大 AI 模型社区，网络效应极强
优势②：元数据丰富，可复现性强，信任度高
优势③：Creator 补偿池机制，激励优质贡献
短板①：支付危机（信用卡全停），付费门槛极高
短板②：监管风险（澳洲封锁，内容合规持续压力）
短板③：平台拆分损伤品牌和用户体验

DESIGN INSIGHTS FOR 千图AI

对千图AI最有参考价值的设计模式

01 · FROM PROMPTBASE
「购买即获取」电商流
参考：PromptBase · Add to Cart → Instant Access
Prompt / Agent / 模板的购买体验应对标电商：清晰定价、一键购买、即时交付、商业授权说明、退款保障。消除"AI 内容不值得付费"的心理阻力。
02 · FROM PROMPTBASE
创作者信任体系
参考：PromptBase · Top Seller 认证 + 星级评分 + 评价数
平台内创作者应有认证等级（新手 / 优质 / Top Seller），内容需显示评分、销量、评价数。用「社会证明」替代平台背书，降低买家决策成本。
03 · FROM PROMPTBASE
80% 创作者分成激励
参考：PromptBase · 0% 直链 / 20% 平台抽成
对比行业标准（苹果30%，普通平台20-40%），提供明显更优的分成比例（建议70-80%），是吸引优质内容创作者快速入场的最有效杠杆。
04 · FROM CIVITAI
丰富元数据 + 可复现性
参考：Civitai · 样本图 + Prompt + 参数 + Seed 全套
千图AI的每个内容单元（模板/Prompt/Agent）需展示完整的"可复现信息"：效果样例 + 使用说明 + 适用场景 + 工具参数。降低用户试用成本。
05 · FROM CIVITAI
创作者补偿池机制
参考：Civitai · 月度补偿池（营收百分比拨款）
除交易分成外，设立「优质内容奖励池」（从平台营收中按月拨款），按内容贡献度（使用量、评分、影响力）分配，激励非商业化的高质量创作。
06 · FROM GPT STORE
分类 + 排行榜发现机制
参考：GPT Store · 8大分类 + Featured + Trending
内容发现必须双轨：编辑精选（Featured，平台信用背书）+ 社区热榜（Trending，真实热度），同时建立清晰的一级 / 二级分类，覆盖不同用户意图。
KEY TAKEAWAYS

核心结论 & 战略启示

市场格局
平台分化加速：Prompt → Agent → Model → Bot，每条赛道独立逻辑
流量壁垒：GPT Store 证明"依附大模型主站"是降维打击，独立平台必须找差异化
Civitai 警示：依赖第三方支付工具有致命风险，支付自主权是生死线
创作者经济：优质分成 + 质量认证 + 数据透明 = 内容飞轮核心
千图AI的护城河 = 图片素材积累 × 创作者信任 × 设计工具链整合
优先抄的3件事
第1优先：PromptBase 的「电商购买流」—— 清晰定价 + 即时交付
第2优先：PromptBase 的「创作者信任体系」—— 认证 + 评分 + 销量
第3优先：Civitai 的「元数据 + 可复现性」—— 每个内容单元信息完整
